# Printer Monorepo

## General
- build and install CLIs: `make install`

## Computer
- path: `computer/`
- build: `cargo build`
- lint: `cargo clippy`
- format: `cargo fmt`

## Codegraph
- path: `codegraph/`
- build: `cargo build`
- lint: `cargo clippy`
- format: `cargo fmt`

## Printer
- path: `printer/`
- build: `cargo build`
- lint: `cargo clippy`
- format: `cargo fmt`
- test: `cargo test`
